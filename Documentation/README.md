## Project Overview
Alnoor Store is a multi-tier e-commerce web application built as a capstone
project for BIT3208. It allows users to browse products, manage shopping carts,
process orders, and administer content through a secure admin panel.

## Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Server:** Apache (XAMPP)
- **Version Control:** Git & GitHub

## Core Features Implemented
- User registration and login with secure password hashing
- Role-based access control (admin and customer)
- Product catalog with search and category filtering
- Full CRUD operations for product management
- Session-based authentication
- Dynamic homepage with live database content
- Responsive design
